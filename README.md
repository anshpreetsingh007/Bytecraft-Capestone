# Markit Roofing Management Platform

A web-based management platform designed for Markit Roofing to help digitize inspection requests, estimates, inventory tracking, and communication between customers, inspectors, and administrators.

## Overview

Markit Roofing currently relies on manual processes for inspections, inventory tracking, and estimating. This project aims to provide a centralized digital platform that improves workflow efficiency and allows employees and customers to access information more easily.

The application provides different features based on user roles:

- Customers can request inspections and view estimates.
- Inspectors can manage assigned jobs, complete inspection forms, and submit reports.
- Administrators can manage users, inventory, schedules, and approvals.

## Features

### Authentication

- User sign-up and sign-in functionality
- Role-based user access:
  - Customer
  - Inspector
  - Admin

### Customer Features

- Submit inspection requests
- View inspection updates
- Access estimates and reports

### Inspector Features

- View assigned inspections
- Complete digital inspection forms
- Upload inspection information
- Submit estimates for approval

### Admin Features

- Manage employees and users
- Track inventory
- Review and approve estimates
- Monitor operations
